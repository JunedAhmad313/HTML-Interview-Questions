Q- What is Xhtml ? Ans- XHTML, which stands for eXtensible HyperText Markup Language, is a type of markup language. It's like an upgraded version of HTML (HyperText Markup Language). XHTML combines HTML with XML (eXtensible Markup Language), forming a kind of application. ⚫️ Key Ponits--- 1- It is the Next Generation of the Html. 2- It is aimed to replace HTML. 3- It is Almost Identical to HTMl. 4- It is a Stricter & clear Version of HTML.
